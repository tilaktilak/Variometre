# variometre
Projet variometre GPS SD Arduino

Le projet intègre plusieurs composants : 
  Un adaptateur SD connecté sur le SPI de l'arduino
  Un écran Nokia connecté aussi sur le SPI 
  Un BMP180 (capteur de pression) en I2C
  Un GPS Neo-6m en USART 
  
Côté Hardware un accu 1s + chargeur USB + Hacheur 5V permettent
l'alimentation du module.
